vamos aprender como criar paginas da internet  

Tags são elementos fundamentais em diversas áreas da programação, especialmente em linguagens de marcação como HTML. Elas são usadas para estruturar, categorizar ou descrever informações, permitindo que sistemas ou desenvolvedores interpretem e manipulem dados de forma eficiente. No HTML, por exemplo, as tags são representadas por palavras-chave envolvidas por colchetes angulares (< >). Elas definem a estrutura e o significado semântico do conteúdo de uma página 

O que podemos desenvolver com ele.

Páginas e sites web completos. Layouts responsivos (que se adaptam a telas de celular, tablet e computador). Aplicações web simples usando HTML, CSS e JavaScript. Templates e protótipos de sites. Formulários, menus de navegação, e galerias de imagens.

Como trabalhamos ou fazemos.

Podemos trabalhar com o Dreamweaver de duas formas: Modo de design (visual) – o desenvolvedor cria a página arrastando elementos (imagens, textos, botões) sem precisar digitar código. Modo de código – o usuário escreve manualmente o código HTML, CSS e JavaScript. Modo dividido – mostra o código e o visual ao mesmo tempo, permitindo ver as mudanças em tempo real. Após criar o site, o Dreamweaver também pode enviar os arquivos para o servidor e publicar o site na internet.

- Explique o HTML

O HTML (HyperText Markup Language) é a linguagem de marcação usada para criar páginas da web. Ele estrutura o conteúdo de um site — define onde ficam os títulos, parágrafos, imagens, links, tabelas, vídeos e outros elementos. O HTML não é uma linguagem de programação, e sim uma linguagem de marcação que usa tags para identificar as partes da página.

- Explique o CSS

; O CSS (Cascading Style Sheets) é a linguagem usada para definir o estilo e o visual de páginas HTML. Enquanto o HTML estrutura o conteúdo, o CSS controla cores, fontes, tamanhos, margens, alinhamentos, bordas e o layout da página

- Explique UTF-8 e para que usamos

: uTF-8 (Unicode Transformation Format - 8 bits) é um padrão de codificação de caracteres. Ele garante que acentos e símbolos especiais (como ç, ã, é, etc.) apareçam corretamente em diferentes navegadores e sistema
